第一步：

安装AHK    https://www.autohotkey.com/

第二步：

双击Bitwig shortcuts for AUP.ahk

即可使用



主要功能：

1.解决部分插件空格键占用问题

2.添加快速加载插件的宏命令



        
快捷键宏：

  Ctrl + 1：暂无

  Ctrl + 2：快速添加Pro C2 Vst(以此类推)

  Ctrl + 3：快速添加Pro Q3 Vst

  Ctrl + 4：快速添加Tool

  Ctrl + 5：暂无

  Ctrl + 6：快速添加Serum Vst

  Ctrl + 7：暂无

  Ctrl + 8：快速添加kontakt 8 Vst

  Ctrl + 9：快速添加Pro R2 Vst
